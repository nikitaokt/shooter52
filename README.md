## требования
игра написана на языке **Python** и требует библиотеку *Pygame*
Для установки *Pygame* необходимо в терминале выпонить
```
pip install pygame
```
or
```
pip install -r requirements.txt
```
## Описание
В игре есть следующие объекты:
1.Player
2.Enemies
3.Asteroids
